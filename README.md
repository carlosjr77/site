# CinePrime

## Descrição do Projeto
Landing page moderna para streaming de filmes e séries.

## Instruções de Deploy
Para fazer o deploy no GitHub Pages:
1. Crie um repositório no GitHub.
2. Faça o push do código para o branch `main`.
3. Vá até as configurações do repositório e ative o GitHub Pages na aba "Pages".

## Recursos Principais
- Interface amigável
- Acesso rápido a filmes e séries
- Sistema de busca
